# 🏢 태평로빌딩 사내 통합관리 시스템

### 📌 개요
이 프로젝트는 인사, 근태, 교육, 급여를 통합적으로 관리하기 위한 **Java 콘솔 기반 시스템**입니다.  
각 관리 기능은 독립된 모듈로 구성되어 있으며, 모든 데이터는 텍스트 파일(`.txt`)로 저장됩니다.  

---

### 📂 기능 구성

| 번호 | 메뉴명 | 설명 |
|------|--------|------|
| 1 | 인사 관리 | 직원 등록, 조회, 검색, 퇴사 등록 |
| 2 | 근태 관리 | 출근/퇴근 기록, 근태 조회 |
| 3 | 교육 관리 | 교육 등록, 수정, 조회 |
| 4 | 급여 관리 | 급여 등록, 조회, 수정, 퇴사 처리 |
| 5 | 종료 | 프로그램 종료 |
